Esercizio su Bootstrap

1 A cosa serve Bootstrap?

Bootstrap serve per creare siti web responsive (cioè che si adattano automaticamente a schermi di dimensioni diverse) in modo semplice e veloce. Fornisce classi CSS già pronte e componenti come bottoni, griglie, navbar, card, modali, ecc.

2️⃣ Cosa devi importare per creare una pagina web con Bootstrap?

Per usare Bootstrap basta importare il suo file CSS e JavaScript (dal CDN o dai file scaricati). Ecco un esempio con il CDN ufficiale di Bootstrap 5:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

3 Cos’è un CDN? Che vantaggi comporta rispetto a scaricare Bootstrap sul tuo sito?

Un CDN (Content Delivery Network) è una rete di server distribuiti nel mondo che forniscono file (come Bootstrap) in modo rapido.
Vantaggi:

  • Caricamento più veloce del sito (i file vengono serviti dal server più vicino all’utente)
  • Non devi salvare i file di Bootstrap sul tuo server
  • Spesso il browser dell’utente ha già i file del CDN in cache

4️⃣ A cosa serve questo attributo del tag meta?

<meta name="viewport" content="width=device-width, initial-scale=1">

Serve a rendere il sito responsive: indica al browser di impostare la larghezza della pagina uguale a quella dello schermo del dispositivo e di usare uno zoom iniziale di 1.

5️⃣ Che differenza c’è tra Container e Container Fluid?

In Bootstrap ci sono due tipi di container:

  • container → ha una larghezza fissa che cambia a seconda della dimensione dello schermo (centrato)
  • container-fluid → occupa sempre il 100% della larghezza della pagina

Puoi provare creando una pagina chiamata BTS_1.1.1.html con: 

<div class="container">
  <h1>Questo è un Container</h1>
</div>

<div class="container-fluid bg-light">
  <h1>Questo è un Container Fluid</h1>
</div>
